# Nixos Multi-Host Configuration

A modular NixOS flake managing multiple machines across desktop and server profiles.

## Hosts

- **Ares** - Niri (Dank Material Shell)
- **Athena** - Niri (Noctalia Shell)
- **Hades** - Headless server profile with Gnome DE for fall-back.

**Managing secrets and SSH keys with [SOPS-NIX](https://github.com/mic92/sops-nix)**
